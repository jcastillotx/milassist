# MilAssist Platform

MilAssist is a comprehensive platform connecting Clients with Virtual Assistants, featuring specialized modules for Travel, Document Review, Research, and Telephony.

## Features
- **Dashboards**: Role-based views for Amins, Clients, and Assistants.
- **Task Management**: Kanban-style task board.
- **Specialized Modules**:
  - Travel Management
  - Document Review Hub (with AI Analysis)
  - Data & Research (with Analytics)
- **Communication Infrastructure**: Call routing and logs.
- **Messaging**: Real-time chat system.
- **Tools**: Visual Page Builder and Advanced Form Manager.

## Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express + Sequelize (SQLite)
